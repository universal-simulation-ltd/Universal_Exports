# Universal Exports

> Open source — self-host free or PRO hosted by UNI SIM.

A fully open-source platform for Export Agreements and trade finance documents.

Two ways to run it:

- **Self-host for free** — fork it, bring your own Supabase, ship it
- **Or use UNI SIM's hosted PRO plan** — skip the setup, [unisim.co.uk](https://www.unisim.co.uk)

Source on [GitHub](https://github.com/jamesmarkeyuk/universal_exports).
